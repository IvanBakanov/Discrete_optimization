Задачи дискретной оптимизации

1. TSP
   - Нумерация городов в коде и в ответе соответствует их порядку во входном файле, начиная с 0;
   - В программе по умолчанию используется пример "TSP_test_200.txt".
2. Knapsack
   - Нумерация предметов в коде и в ответе соответствует их порядку во входном файле, начиная с 0;
   - В программе по умолчанию используется пример "KS_test_100.txt".
